# quiz4studyW

Lightweight study quiz web app for quick self-check sessions.

## Demo
![Demo](https://raw.githubusercontent.com/Pancake2021/Pancake2021/main/assets/cloud-drift.gif)

## How to run
```bash
# option 1: open index.html
# option 2: run local static server
python -m http.server 8080
```

## Results
- Fast quiz loop with simple deployment.
- Good base for adding question pools and scoring analytics.
- Student-friendly UX and lightweight stack.
